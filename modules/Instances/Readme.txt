<h3>Crm Analyser Plugin</h3>
<strong>Hello vTigers and vTigresses,</strong>

Crm Analyser connects differnet crm instances in one instance.
It display records of each module in tabs.
Created one custom module as Instances to store the db name, host name, username and password.
DetailView Sidebarwidget(Analyser) in Instances module will display the CRM Analyser Data.
